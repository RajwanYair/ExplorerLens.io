## Quick Install Path Reference

```text
# User plugins directory
%LOCALAPPDATA%\ExplorerLens\Plugins\

# System-wide (admin required)
%ProgramFiles%\ExplorerLens\Plugins\
```

Or register via **LENSManager.exe** → Plugins → Install From File.
